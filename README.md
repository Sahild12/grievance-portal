# Citizen Grievance Portal

A simple and effective web application for citizens to file and track grievances.

**Live Demo:** [https://jansevaa.netlify.app/](https://jansevaa.netlify.app/)

## Features

*   **Citizen Dashboard:** View and manage your grievances.
*   **Submit Complaints:** Easily file new grievances.
*   **Track Status:** Keep track of the status of your submitted complaints (Pending, Resolved).
*   **Official Dashboard:** A dedicated interface for officials to manage and address complaints.

## Tech Stack

*   **Frontend:** HTML, CSS, JavaScript
*   **Backend:** Node.js, Express.js
*   **Database:** MongoDB

## Getting Started

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/grievance-portal.git
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd grievance-portal
    ```
3.  **Install dependencies:**
    ```bash
    npm install
    ```
4.  **Start the server:**
    ```bash
    npm start
    ```
5.  Open your browser and visit `http://localhost:3000`.
